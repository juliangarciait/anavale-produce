All in One Cancel Sale, Purchase, Picking and Invoice

This app helps user to cancel all orders in odoo like sale, purchase, invoice, and picking and received and delivered product quantity will reverse. This module will allow user undo records in some cases in Odoo ERP.

Cancel Sale Order

User can cancel sale order, on canceling sale order related delivery orders and invoices will cancelled automatically.

Cancel Purchase Order

User can cancel purchase order, on canceling purchase order picking order and bill related to order will cancelled automatically.

Delete Stock Journal Entries

If product inventory valuation is automated then stock journal entries will get deleted while canceling picking.
