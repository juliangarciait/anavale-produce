Profit & Loss + Balance Sheet Analytic Tag Groups

Geminate comes with a feature of grouping based on analytic tags on financial reports like balance sheet and profit & loss. it will tabularize all financial data from different charts of accounts under each analytic tags based on journal items. it will enable grouping only in case there is data available in that period / duration or year.

By that way, you can easily visualize the total amount utilized under that analytic tags within the individual chart of the account per duration.







Editions Supported

1. Enterprise Version

Note: This module is developed, designed and tested on odoo vanila.

Available Key Features

  • Enable grouping based on analytic tags for financial reports like balance sheet and profit & loss.
  • Tabularize all financial data from different charts of accounts.
  • Perform grouping on data available within period / duration or year.
  • Easy to visualize total amount utilized under individual analytic tags.
  • Toggle button to enable / disable analytic tags with different highlighted color.
  • Easy to use and quick to apply.
